Xml 

<?xml version="1.0" encoding="utf-8" ?>
<user type="client">
	<id> 0 </id>
	<fname> fname1 </fname>
	<lname> lname1 </lname>
	<contact>
		<address> Street 1 </address>
		<address> Street 2 </address>
		<website> user.com </website>
	</contact>
	<partof db="users" />
</user>

Xpath for <fname> fname1 </fname> is /user/fname ( absolute )

Xpath for <address> Street 1 </address> is contact/address[1] ( relative if in <user> )

Parsed as with Html.

Has operators for selectors.
